Your News Website on Autopilot. It is using the newsapi.org API to fetch the News from more than 30 000 News Sources.

== Description ==
With the News API for Wordpress you can have in minutes a modern and profesional NEWS Website, updated automaticaly with the latest NEWS from the choosed Topic or Country.
News API made for newsapi.org API. A powerful news, modern, trending website.
Topic/ Country news with the news API from newsapi.org. Please check the “News API Terms of Service” at https://newsapi.org/terms.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/wpm-news-api` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Admin Menu->WP news API->Settings screen to configure the plugin
4. Find more about the WP news API plugin at `https://docs.wpmagic.cloud`
